СПРАВОЧНИК ТЕГОВ
Тэг Атрибуты Что делает От какого слова образовано
h1, h2, h3... h6 Заголовок heading
p Абзац текста paragraph
img src, height, width Вставляет картинку image, source (путь к файлу)
a href Вставляет ссылку anchor, hypertext referense
table Таблицу Таблица
thead Шапка (заголовок таблицы) table head (the ад)
tbody Содержимое таблицы table body (столовое тело)
th Ячейка в шапке таблицы table heading
td Просто ячейка table data (столовая дата)
ul Маркированный список (то'сть кружочки) unordered list (непорядочный лист)
ol Нумерованный список ordered list (порядочный лист)
li Элемент списка list item (листовая вещь)
Справочник css свойств
Свойство Значения Что делает
color Цвета (red, blue, yellow...) Изменяет цвет элемента и текста а также границы
background-color red, blue, yellow... Цвет фона элемента
text-align left, right, center Выравнивание текста
text-decoration _-‾ underline, overline, line-through, none Подчёркивание, надчёркивание, перечёркивание, отмена -чёркивания
font-size От 1 до неограниченного количества (16px - стандарт) Размер шрифта
font-style normal, italic Норм, курсив
font-weight normal, bold Норм, жирный (регулируется)
border-color Tilda colors (набрать в Инете) Цвет границы
border-width 1px, 2px... Ширина границы
border-style solid, dashed, dotted, double Тип линии
border-collapse collapse, separate Убирает/добавляет пространство между ячейками
border-radius Либо в px, либо в % Закругляет уголки блока (картинки)
list-style-type disk, circle, square, none Символы маркированного списка
margin В px или в % Внешний отступ (отталкивает элементы друг от друга)
padding В px или в % Внутренний отступ (внутренние поля - увеличивает размер элемента)
position relative, absolute, static, fixed Позиционирование (двигать блоки)
